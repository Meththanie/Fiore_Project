# FIORE Project  
**AI-Powered Event Planning Solution**

## Overview  
FIORE is an innovative web application that leverages artificial intelligence to simplify and enhance event planning. Whether you're organizing a wedding, birthday party, or any other gathering, FIORE ensures seamless coordination, personalized recommendations, and an exceptional user experience.  

## Key Features  
- **Event Personalization**: Tailored suggestions for themes, venues, and vendors.  
- **AI-Driven Recommendations**: Smart algorithms to match preferences and budgets.  
- **User-Friendly Interface**: Intuitive design for easy planning and management.  
- **Comprehensive Planning**: Everything you need in one place, from guest lists to itineraries.  

## Technologies Used  
- **Frontend**: React.js  
- **Backend**: Node.js and Express  
- **AI Frameworks**: TensorFlow/PyTorch  
- **Database**: MongoDB  
- **Hosting**: AWS  

## How to Run This Project Locally  

### Prerequisites  
- Ensure you have the following installed:  
  - [Node.js](https://nodejs.org/) (LTS version recommended)  
  - [Git](https://git-scm.com/)  

### Steps  
1. **Clone the Repository**  
   Open your terminal and run:  
   ```bash  
   git clone https://github.com/Sasmitha000/Project-FIORE.git
   ```  

2. **Navigate to the Project Directory**  
   ```bash  
   cd FIORE  
   ```  

3. **Install Dependencies**  
   Run the following command to install all required packages:  
   ```bash  
   npm install  
   ```  

4. **Set Up the Environment Variables**  
   Create a `.env` file in the root directory and add the necessary configuration details (e.g., database connection strings, API keys).

5. **Start the Development Server**  
   Run the following command to start the application:  
   ```bash  
   npm start  
   ```  

6. **Access the Application**  
   Open your web browser and go to:  
   [http://localhost:3000](http://localhost:3000)  

   The application should now be running on your local machine.
